# Roshan Khata

Digital bahi khata app for shopkeepers, agri-dealers & wholesalers.
A project of **Innovation-313**.

Kotlin-native Android app featuring a Zakat calculator, Qarz-e-Hasna (interest-free
loan) tracking, weight/quantity-based agri-ledger, and an offline-first design —
built to serve everyone from a small kirana shop to a wholesale distributor.

## Status
🚧 Early scaffolding — Stage A (MVP) feature build in progress. Not yet published.
